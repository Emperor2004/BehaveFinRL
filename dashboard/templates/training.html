Kahneman-Tversky Prospect Theory (1979)

Standard Reinforcement Learning models assume rational economic actors (maximize raw expected returns). In contrast, BehaveFinRL embeds human behavioral psychology directly into the agent's reward channel.

// Value Function Equations

if x ≥ 0: v(x) = xα (Gains — Concave)

if x < 0: v(x) = -λ · (-x)β (Losses — Convex)

// Empirical Parameters

α = β = 0.88, λ = 2.25 (Baseline Loss Aversion)

Key Properties

  • Reference Dependence: Outcomes are judged relative to entry capital (P&L return %) rather than absolute wealth levels.
  • Loss Aversion: The utility curve is steeper for losses than gains. The agent experiences a $1,000 loss 2.25 times more intensely than a $1,000 gain.
  • Diminishing Sensitivity: Exponential exponents (0.88) lead to a concave gain curve and convex loss curve, mimicking actual investor preference limits.
Reward shaping forces the PPO policy to act as a highly risk-aware portfolio manager.
